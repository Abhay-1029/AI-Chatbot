# AI Chatbot

A simple AI chatbot built with Python and OpenAI API.

## Features

* Interactive command-line chatbot
* Uses OpenAI GPT model
* Maintains conversation history

## Installation

```bash
pip install openai
```

## Setup

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key_here
```

## Run

```bash
python main.py
```

## Tech Stack

* Python
* OpenAI API
